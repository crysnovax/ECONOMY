# ECONOMY
Introducing CRYSNOVA-Ai, A WhatsApp bot that automates interactions on WhatsApp by executing predefined commands or responding to user inputs. It can handle tasks like sending messages, sharing media, and managing group activities, offering convenience and efficiency for users and businesses.

# 🏦 CRYSNOVA Economy API

Realistic virtual economy engine for WhatsApp bot.

## Endpoints (27+)
| Category | Endpoints |
|----------|-----------|
| Banking | balance, deposit, withdraw, pay, gift |
| Jobs | work, fish, mine, hunt, beg, daily, weekly |
| Crime | rob, crime, drugs, attack |
| Progression | training, levelup, travel, faction, loan, profile |
| Shop | shop, buy, sell, inventory |
| Admin | admin/stats, admin/add-money, admin/remove-money, admin/reset |

## Admin Dashboard
Visit `/admin` to login. Default password set via `ADMIN_PASSWORD` secret.
